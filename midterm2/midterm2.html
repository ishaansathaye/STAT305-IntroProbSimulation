Relationship between Poisson and Exponential Distributions

Recall the result: If the number of occurrences per time interval follows a Poisson() distribution, then the length of time between occurrences follows an Exponential() distribution.

  • Let be the number of customers arriving at a bank branch in one hour, where Poisson(20).

  • Let be the time (in minutes) between consecutive arrivals. Since there are on average 20 arrivals per hour, the per-minute rate is 20/60, and so Exponential(20/60).

  • Simulate one hour of customer arrivals for this bank by generating inter-arrival times from an Exponential(20/60) distribution. Stop when the total time exceeds 60 minutes, and count the total number of customers arrived within this hour.

    # A safe upper bound for maximum number of customers within an hour
upper <- 1000  
y <- numeric(upper)  # Empty vector to store inter-arrival times
count <- 0  # Counter for number of arrivals

for (i in 1:upper) {
  y[i] <- rexp(1, rate = 20/60)  # Generate an inter-arrival time (in minutes)
  if (sum(y[1:i]) >= 60) {  # If total time exceeds or equals 60 minutes, exit the loop
    break
  } else {
    count <- i  # Record the current number of arrivals
  }
}
# 'count' now represents the number of customers that arrived within 60 minutes.
count
    ## [1] 16

  • Repeat above code 100,000 times in order to simulate 100,000 realizations of using the inter-arrival times only (not rpois()). This will require a for loop within a for loop, which is a common coding practice. Make sure to use a different index for the outer loop, say j, since we used the index i in code given above, which will serve as the inner loop. Also, use set.seed(1) so that we have the same simulation results. Save 100,000 realizations in a vector, say x.

    set.seed(1)

n_sim <- 100000  # Number of simulations
x_sim <- numeric(n_sim)

for (j in 1:n_sim) {
  upper <- 1000  # Safe upper bound
  y <- numeric(upper)
  count <- 0
  for (i in 1:upper) {
    y[i] <- rexp(1, rate = 20/60)
    if (sum(y[1:i]) >= 60) {
      break
    } else {
      count <- i
    }
  }
  x_sim[j] <- count
}

# Check the first few simulated values
head(x_sim)
    ## [1] 16 24 20 11 19 26
